Overview

Welcome to the official StyleSite specifications guide. StyleSite is an atomic, component-driven CSS layout library engineered to cleanly extract typography arrays entirely out of rigid background boxes, side-by-side grids, and blocking card layouts. It enforces an absolute one horizontal element per line stack rule to completely erase side-by-side text compression and content layout overlaps on any device.

Structural Philosophy

Traditional UI frameworks force your site data into tight columns that break and squeeze on smaller screen viewports. StyleSite structural properties stack your code elements linearly on a vertical axis, organizing elements cleanly on top of an interactive dark aurora canvas with high-legibility spacing and responsive typography.

Installation

// Plugs seamlessly into your web project layouts

CDN Production Delivery

To implement StyleSite's styling rules instantly across your public web endpoints, grab and append the compiled production stylesheet link right inside your workspace document's primary <head> module:

<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/sahirpunjwani/stylesite@main/index.css">

NPM Integration

If you prefer installing packages straight into your active node configuration workflows, track and bundle the core package module using the command line profile:

npm install @stylesite/core

Usage Guide

// Semantic syntax maps your layout properties natively

1. Root Workspace Setup

Initialize and normalize your document background instantly by applying the .ss-workspace style class right onto your main <body> wrapper element:

<body class="ss-workspace">
    <!-- Content segments deploy inline -->
</body>

2. Direct Single-Row Layout Stacking

To layout bare text content rows cleanly without enclosing data arrays inside container boxes, pass components through the .ss-row-stack structural frame wrapper block:

<div class="ss-row-stack">
    <div class="ss-unboxed-content ss-motion-reveal">
        <h3>Component Heading</h3>
        <p>This text displays bare, container-free, and styles itself automatically...</p>
    </div>
</div>

3. Widescreen Graphic Title Backdrops

To launch elegant title header visuals backed by premium background artwork patterns, embed the standalone .ss-hero-backdrop layout component inside your workspace blocks:

<div class="ss-hero-backdrop ss-motion-reveal" style="background-image: url('your-backdrop.jpg');">
    <h2>Workspace Title Row</h2>
    <p>// Engineering Track Metrics Subtitle</p>
</div>
